CSS

Introduction

CSS stands for Cascading Style Sheets and is used for controlling the style and layout of your HTML. CSS allows for you to access very specific elements, classes, and ids, to control even the most nested element in your webpage.

CSS can be used directly inside of your HTML or it can be outside on a separate file and have it's directory linked to your HTML.

Basics

Initialization and Syntax

CSS follows a format called declarative syntax. Declarative syntax follows by starting with a selector which could be either a tag, class, or an id.

Inside the selector is where you would control the selectors properties such as background, color, margin, padding, etc. These all allow for formatting and positioning on your webpage.

Selectors

CSS has many different types of selectors that are used to target a specific tag, id, or class. This will allow you to precisely set the necessary attributes wether you want to access the 5th p tag inside of 10 total p tags or you want to access a whole class.

To access a broad tag throughout your page you can simply call the tag name div or p and then supply them with the necessary attributes.

To call a specific id you use "#" hashtag and the name of the id. For example #someId and then you can continue with the brackets and attributes. It doesn't matter where the id is located in the HTML wether its deeply nested or not, all you need to do is correctly call its name.

To call a specific class you use the "." period and then the name of the class. The same rules apply as calling ids just make sure you are using the correct symbols and names.

Properties and Values

Inside a selector block are the properties applied to the selected elements, IDs, or classes.

On the left side is the property/attribute of the element and on the right is the value assigned to that property.

After each property call you will use a semicolon to end that line, either to finish setting properties or to continue to the next property you want to set.
