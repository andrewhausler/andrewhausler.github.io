CSS

Introduction

CSS stands for Cascading Style Sheets and is used for controlling the style and layout of your HTML. CSS allows for you to access very specific elements, classes, and ids, to control even the most nested element in your webpage.

CSS can be used directly inside of your HTML or it can be outside on a separate file and have it's directory linked to your HTML.

Basics

Initialization and Syntax

CSS follows a format called declarative syntax. Declarative syntax follows by starting with a selector which could be either a tag, class, or an id.

Inside the selector is where you would control the selectors properties such as background, color, margin, padding, etc. These all allow for formatting and positioning on your webpage.
