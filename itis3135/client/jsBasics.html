JavaScript

Introduction

JavaScript is a scripting language that allows you to work with static HTML files to make it more interactive.

With JavaScript you can simply add the path to your script file and include that in the HTML. Then you can immediately start calling and assigning all tags, classes, and elements.

Basics

Initialization and Syntax

JavaScript is very similar to other languages out their like Python, Java, etc. It has very similar data types and syntax as well as having re-assigning variables and non-re-assigning variables.

JavaScript also includes its own implementation of conditionals, loops, objects, and arrays, making it a flexible and powerful language for various programming tasks.
