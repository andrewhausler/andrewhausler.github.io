About

Why

This website was assigned to develop a website by gathering information from a client. The design, layout and the structure of this website is all designed by the information that I had received from my client.

It was also built to represent to importance in web-development designing a website while following by your clients wants and needs.

Purpose

The purpose of this website is for Niners that are unsure of what path the want to follow. This will allow any Niner to quickly grab a grasp of knowledge by trying out these important front-end languages.

Each page contains it's own information of a language. Each language contains the basics of each language with examples as well.
