HTML

Introduction

HTML stands for Hypertext Markup Language. It is used in all web application you see whenever you are viewing the web.

HTML by itself is a static markup language, meaning that all content remands unchanged and whatever data is on that file remains and doesn't change. The only way it will change is if you send a new request to the webserver that is running on the site you are using.

Basics

Initialization and Syntax

HTML is much more laid back with structuring a webpage but the browser that the webpage will run is not. It's important to follow the correct structure in developing a webpage to run as it should.

Every HTML page should always begin with <!DOCTYPE html>. The doctype allows your browser to know you want to use up-to-date HTML and to not use any legacy versions.

These brackets <> are called angle brackets. Angle brackets are used to define the HTML tag you want to use when laying out your HTML. There are many different types of tags headings, paragraphs, images, etc. They all allow the browser to identify your layout.

Structure

After the DOCTYPE you can create the HTML tag <html lang="en">. The HTML tag is the root element in creating an HTML page.

The HTML tag also allows the browser to recognize it's working with an HTML file.

The beginning HTML tag <html> represents the start of the HTML page and the ending tag </html> represents the end of the HTML page.

Finally setting the language property of the HTML <html lang="en"> is recommended as it helps search engines understand the page's language as well as working with translation tools.

Metadata

The next tag after the HTML tag is the head tag. The head provides data and information to the browser and search engines about the document.

Inside of the head element it can contain a meta, title, favicon, link, etc. The meta tag is a void tag that takes the character encoding property <meta charset="UTF-8">.

The next meta is used for setting the view of the page. Webpages on many different devices can come out looking very different by the sizes of the device screen. When setting the viewport according to the devices screen size, allows for easier viewing and reading.

The title tag is used for setting the name of the page. This name you set will be viewable on the tab of the page.

Another commonly used tag in the head element is the link tag <link>. This tag is a void tag and used to link external resources such as linking you CSS stylesheet.

Body

The body is where the actual information on your webpage goes.

There are three important parts that should be inside of the body. There should always be a header, main, and a footer.

The header is the upper most part of the page. This part of the page should consist of some sort of title of the page and a navigation part connected all the other pages.

In the middle of the page there should be the main information/content meant for that page.

Lastly, the final piece is the footer which can be used for providing external sources as well as giving credit to developers, websites, servers, or any application thats a part of your website.
