HTML

Introduction

HTML stands for Hypertext Markup Language. It is used in all web application you see whenever you are viewing the web.

HTML by itself is a static markup language, meaning that all content remands unchanged and whatever data is on that file remains and doesn't change. The only way it will change is if you send a new request to the webserver that is running on the site you are using.

Basics

Initialization and Syntax

HTML is much more laid back with structuring a webpage but the browser that the webpage will run is not. It's important to follow the correct structure in developing a webpage to run as it should.

Every HTML page should always begin with <!DOCTYPE html>. The doctype allows your browser to know you want to use up-to-date HTML and to not use any legacy versions.

These brackets <> are called angle brackets. Angle brackets are used to define the HTML tag you want to use when laying out your HTML. There are many different types of tags headings, paragraphs, images, etc. They all allow the browser to identify your layout.
